**Diese Geschäftsbedingungen (die „Vereinbarung“) regeln Ihre Teilnahme am Marketplace-Programm von GitHub. Indem Sie sich für die Teilnahme am Marketplace anmelden oder unten auf „Ich akzeptiere“ klicken, bestätigen Sie, dass Sie diese Vereinbarung verstanden haben und alle darin enthaltenen Bestimmungen und Bedingungen akzeptieren. Wenn Sie diese Vereinbarung im Namen eines Unternehmens oder einer anderen juristischen Person abschließen, versichern Sie, dass Sie rechtlich befugt sind, die juristische Person an diese Vereinbarung zu binden; in diesem Fall bedeutet „Sie“ die juristische Person, die Sie vertreten.**

Gültig ab: 27. Mai 2025

[1. DEFINITIONEN](#1definitions)
----------

Feste Begriffe, die in dieser Vereinbarung verwendet und hierin nicht definiert werden, haben die Bedeutung, die in den GitHub Nutzungsbedingungen unter [https://docs.github.com/articles/github-terms-of-service](/de/site-policy/github-terms/github-terms-of-service) angegeben ist. Diese Nutzungsbedingungen werden als „Bedingungen“ oder „ToS“ bezeichnet.

„Aktion“ bezeichnet einen Workflow, mit dem Sie eine benutzerdefinierte Logik implementieren können, um eine bestimmte Aufgabe mit GitHub auszuführen, ohne eine Anwendung erstellen zu müssen.

„Markenmerkmale“ bedeutet die Marken und ähnliches geistiges Eigentum jeder Partei.

„Entwickler“ oder „Sie“ bezeichnet das Unternehmen oder die Person, die die Entwickleranwendung erstellt hat.

„Entwickleranwendung“ oder „Entwicklerprodukt“ bezeichnet die Software, die damit verbundenen Inhalte und andere digitale Materialien, die von Ihnen zur Verwendung in Verbindung mit GitHub erstellt wurden und über den Marktplatz zugänglich sind. Dazu gehören Aktionen, GitHub Copilot-Erweiterungen und andere Anwendungen oder Produkte, die mit Technologie generativer künstlicher Intelligenz (AI) arbeiten oder interagieren.

„Endbenutzer“ bezeichnet jede Person, Firma oder andere juristische Person, die Lizenzen für ein Entwicklerprodukt über den GitHub Marketplace erwerben wird.

„GitHub-API“ bezeichnet jede proprietäre Anwendungsprogrammierschnittstelle, die von GitHub entwickelt wurde, und umfasst die GitHub Copilot-API. Der Zugriff auf eine GitHub-API und ihre Nutzung wird durch die Nutzungsbedingungen geregelt.

„GitHub Marketplace“ oder „Marketplace“ bezeichnet die von GitHub betriebene proprietäre Online-Marktplatzseite, auf der Entwicklerprodukte an Endbenutzer geliefert werden können.

„Auflistung“ bezeichnet den Inhalt, der für die Auflistung des Entwicklerprodukts auf dem GitHub Marketplace bereitgestellt wird.

„Steuern“ bedeutet jede Art von Steuer, unabhängig von ihrer Herkunft, einschließlich Zinsen oder Strafen.

„Token“ bedeutet, dass Entwickler die Anmeldeinformationen abrufen müssen, um auf eine GitHub-API zuzugreifen.

„Nutzungsdaten“ bezeichnet Marketplace-bezogene Daten, die in Verbindung mit der Nutzung von GitHub Marketplace durch den Endbenutzer und der Lizenzierung von Entwicklerprodukten generiert werden, einschließlich, aber nicht beschränkt auf Nutzungsstatistiken und aggregierte Verkaufsdaten. Nutzungsdaten umfassen keine Bank- und Zahlungsinformationen und schließen diese ausdrücklich aus.

Der Entwickler erklärt sich damit einverstanden, den GitHub Marketplace ausschließlich für Zwecke zu nutzen, die durch diese Vereinbarung zulässig sind. Entwickler sind allein für die Entwicklerprodukte verantwortlich.

Wenn Entwickler die Verteilung eines Entwicklerprodukts auf Marketplace beenden, beendet GitHub die Verwendung der Markenmerkmalen der nicht mehr vorhandenen Produkte.

**2.4** Der Entwickler muss seinen Entwicklerprodukten eine separate Endbenutzer-Lizenzvertrag („EULA“) hinzufügen, der die Rechte des Endbenutzers an den Entwicklerprodukten regelt. GitHub übernimmt keinerlei Verantwortung oder Haftung im Rahmen von EULAs.

**2.5** Mit Ausnahme der in dieser Vereinbarung gewährten Lizenzrechte (a) behält der Entwickler alle Rechte an den Entwicklerprodukten; (b) behält jede Partei alle Rechte, die sie unabhängig von dieser Vereinbarung hat, einschließlich der Rechte nach dem US-Urheberrechtsgesetz oder ähnlichen Gesetzen anderer Gerichtsbarkeiten; und (c) besitzt jede Partei alle Rechte, Titel und Anteile an ihren jeweiligen Markenkennzeichen.
