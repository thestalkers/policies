[概要](#overview)
----------

お客様のプライバシーは GitHub (以下「当社」または「GitHub」といいます) にとって重要です。 当社は、すべての個人のプライバシー権を尊重し、適用される法令に従って、責任を持って個人データを取り扱うことに努めています。 本プライバシー通知は、当社がデータ収集時にお客様に提供する場合がある補遺およびその他の通知と共に、GitHub がお客様に関して処理する個人データ、当社によるこの個人データの使用方法、およびこの個人データに対するお客様の権利について説明するものです。

本プライバシー通知は、候補者としてのお客様の個人データの取り扱いに適用されることにご注意ください。

本通知は、GitHub コンシューマー製品の消費者としてのお客様による使用または GitHub との候補関係外での使用については扱っていません。 GitHub コンシューマー製品には、サービス、Web サイト、アプリ、ソフトウェア、サーバー、およびデバイスが含まれる場合があります。 消費者としてのお客様による GitHub 製品の使用について扱っている GitHub のデータ収集プラクティスの詳細については、[GitHub プライバシー ステートメント](https://site-policy/privacy-policies/github-general-privacy-statement)を参照してください。

本通知は、雇用のため、何らかの利益のため、または特定の状況における特定の待遇のために、明示的または黙示的な約束または契約を形成することを意図したものではなく、そのように解釈されてはなりません。 本通知のいかなる規定も、当社の法的義務を遵守する目的で、または地域の法的要件への準拠を条件として、不正行為または会社のポリシーもしくは法律への違反の疑いを調査するために、GitHub が候補者データを処理する能力を妨げるものとは解釈されるべきではありません。

GitHub による個人データの処理は、すべての場合において、適用される地域の法令の要件、内部ポリシーの要件、および該当する場合または適切な場合は労働者の代表者との協議要件の対象となります。 本通知がお客様の管轄区域における地域の法令と矛盾する範囲においては、地域の法令が適用されます。

### [当社が処理する個人データ](#personal-data-that-we-process) ###

(これには、お客様が当社に提供したデータ、当社がお客様に関して収集または生成したデータ、または当社がお客様に割り当てたデータが含まれる場合があります。)

当社は、当社の事業運営において、お客様に関するさまざまな種類の個人データを収集、使用、および保存 (以下、総称して「処理」といいます) します。 お客様が候補者である場合、当社が処理する個人データの種類は一般に、当社が GitHub のキャリア機会に関してお客様とやり取りする必要がある内容、お客様による GitHub における特定の職務への応募についての検討 (候補者のスクリーニング、面接のスケジュールと管理、合法的なバックグラウンド スクリーニング、および当社での雇用のオファーを受け取って受け入れる場合は GitHub へのお客様のオンボーディングが含まれます) に限定されます。

当社が処理する個人データには、以下が含まれる場合がありますが、これらに限定されません。

**名前と連絡先データ**。 お客様の氏名、従業員識別番号、メール アドレス、郵送先住所、電話番号、写真、受益者および緊急連絡先の詳細、およびその他の類似する連絡先データ。 また、お客様は、個人のメール アドレスおよび/または携帯電話番号などの追加の連絡先情報を GitHub に提供することもできます。

**人口統計データ**。 お客様の生年月日および性別、ならびに人種的および民族的出自、宗教、宗教上もしくは哲学上の信念、労働組合のメンバーシップ、またはお客様の健康、障がい、性的指向、性同一性、およびトランスジェンダーの状態に関する情報を含む、より機微な個人データ (特別な種類のデータとも呼ばれます)。 また、当社は、お客様の子供の有無や軍歴についても伺う場合があります。

**国が提供する識別情報**。 お客様の国民 ID/パスポート、市民権の状態、居住および労働許可の状態、社会保障番号、またはその他の納税者/政府発行身分証明書番号。

**雇用の詳細**。 役職/職位、アカウント資格情報、オフィス所在地および/またはリモート勤務地、雇用契約、内定通知、雇用日、退職日、実績履歴および学業記録、労働時間、バッジ活動情報、トレーニング記録、休職、病欠、ならびに休暇/休日記録。

**配偶者/パートナーおよび被扶養者の情報**。 お客様の配偶者と被扶養者の氏名、生年月日、および連絡先の詳細。

**バックグラウンド情報**。 お客様の学術資格および職業資格、学歴、履歴書、信用履歴および犯罪記録データ。

**動画、音声、および画像**。 当社は、地域の法令の要件、内部ポリシーの要件、および労働者の代表者との協議要件 (適切な場合) に従って、お客様の動画、音声、および画像データを収集および使用する場合があります。

**財務情報**。 お客様の銀行口座の詳細、税情報、給料、年金口座情報、会社の手当、ならびに給与、税、福利厚生、および株式報奨とインセンティブ報酬を管理するために必要なその他の情報。

**推測**。お客様の個人的特性 (職歴や経験など)、能力、および適性を反映したプロファイルを作成するための推測。

**個人データのソース**

当社は、お客様から (直接的または間接的に) 個人データを収集することに加えて、お客様に関する個人データを、以下を含む、お客様以外のデータ ソースから取得します。

* 当社がお客様に関して処理する個人データからお客様に関して生成または派生した推測データ。
* 以前の雇用主や現在の雇用主からの推薦を含む、お客様が提供する推薦またはお客様に関する推薦。
* 第三者または公開のソース (たとえば、採用を目的とした、LinkedIn プロファイルなどの、公開のプロフェッショナル ネットワーキング ソースからの情報)。
* 当社のベンダーおよびその他のプロバイダー (当社が法律で許可されている範囲で、お客様の過去の学歴、雇用歴、信用履歴、および/または犯罪歴に関する情報を当社に提供する第三者ベンダーを通じて、合法的なバックグラウンド スクリーニングを実施する場合など)。

### [当社が個人データを処理する理由](#why-we-process-personal-data) ###

当社は、以下に規定する目的で、お客様の個人データを収集および処理します。当該目的には以下が含まれる場合がありますが、これらに限定されません。 お客様の個人データが要求したときに提供されない場合、当社がこうしたタスクを実行すること、および/または当社の法的義務を遵守することができなくなる可能性があります。

**採用および雇用**。 当社による採用および雇用のため。これには、GitHub での雇用機会に関するお客様とのやり取り、お客様による特定の職務への応募についての検討、候補者のスクリーニング、面接のスケジュールと管理、合法的な身元調査とバックグラウンド スクリーニング、内定通知の準備が含まれます。

**給与、報酬、および会計**。 当社の給与および報酬活動のため。これには、税の報告、株式報奨とインセンティブ報酬の管理、報酬の管理、経費の払い戻し、その他の会計タスクが含まれます。

**福利厚生の登録**。 福利厚生の登録に関連する活動のため。

**キャリアのプランニングおよび開発**。 キャリアのプランニングおよび開発に関連する活動のため。これには、お客様の GitHub におけるキャリア機会の評価および提供が含まれます。

**機会均等の評価および便宜**。 当社の雇用機会均等、多様性、インクルージョンおよびアクセシビリティ プログラムを管理および評価するため。これには、平等な待遇と機会の監視および確保、仕事に関連する便宜または調整の提供、当社のグローバル機会均等イニシアティブが含まれます。

**法令遵守およびポリシー遵守の管理および実施**。 当社の法令遵守およびポリシー遵守に関するプログラムおよび要件を管理および実施するため。これには、当社の契約、政府機関の許可、および適用される GitHub ポリシーに基づく当社の義務の履行、必要な記録の保持、司法上の承認に基づく個人データの収集または開示 (米国または外国の管轄区域における裁判所命令、行政または訴訟手続き、または公的機関によるその他の合法的な要請など)、合法的なバックグラウンド スクリーニングの実施、法令 (最低賃金、労働時間、税、健康と安全、差別防止法、政府機関への報告義務、グローバル移住、内部通報手続き、データ主体の権利など) の遵守、ならびに GitHub の法的権利の侵害、行使、または防御の可能性の調査 (法的助言の追求、不正行為への対抗、従業員その他の生命と安全の保護を含む) が含まれます。

**科学研究**。 科学研究に関連する活動のため。これには、適切な技術的かつ組織的な管理 (データの匿名化および集計、当社のプライバシー基準への準拠、倫理および準拠性レビューの実施など) を条件として、公共の利益のためと考えられる研究および一般化可能な知識に貢献しようとする研究が含まれます。

**パーソナル化**。 お客様の好みを理解してお客様の候補者エクスペリエンスを向上させるなどのパーソナル化活動のため。これには、お客様の好みや設定を保存および尊重すること、お客様による当社のシステムやアセットへのサインインまたは他の方法でのアクセスや使用を可能にすることが含まれます。

**AI および自動化された意思決定**。 GitHub は当社の製品において AI 対応のエクスペリエンスを有効にしているため、チャットボット機能、要約機能などの AI 機能を含む、GitHub テナントにデプロイされている一定の機能やエクスペリエンスを支援するために、お客様の個人データも AI によって処理される場合があります。 許容される場合、本条で説明する業務目的でお客様のデータを処理するために、人工知能 (AI) テクノロジと機械学習 (ML) テクノロジが使用されることもあります。 GitHub によるお客様のデータの処理は、責任ある AI に向けた当社の取り組みに準拠します。 また、当社は、自動化された意思決定システムを使用して、傾向や停止の特定、当社のシステムとデータの監視およびセキュリティ保護、チャットボットやその他の対話型サービスの提供、ならびに当社のポリシーと手順の遵守の監視を行う場合があります。

**一般的な事業運営**。 当社の一般的な事業運営のため。これには、事業の一般的な管理、当社のビジネス アプリケーションおよびシステムの実装と管理、コミュニケーションとコラボレーションの促進が含まれます。

当社は、上記の目的のために、さまざまなソースのデータを組み合わせることがあります。

また、当社は、お客様との業務関係を管理するために必要に応じて、適用される法令で許容されているその他の正当な業務目的で個人データを処理する場合があります。

法律で義務付けられている場合、当社は上記の使用についてお客様の同意を求めます。お客様の同意を求める場合、当社は、お客様の同意が情報に基づき、自発的に行われ、お客様がご自身の同意の保留または取り消しの決定によって悪影響を受けないことを確実にします。

### [目的の変更](#change-of-purpose) ###

当社は、お客様の個人データを、別の互換性のある目的に合理的に必要であり、さらなる処理のための法的根拠がある場合を除き、収集された目的でのみ使用します。 たとえば、GitHub での職務の候補者の採用に関する当社の正当な利益に依拠して、求人の調査中にお客様から提供される個人データを処理する場合があります。 ただし、お客様が職務に応募して採用された場合、当社はお客様との雇用関係を開始する目的でお客様の個人データを処理することがあります。

### [個人データを開示する方法および理由](#how-and-why-we-disclose-personal-data) ###

GitHub は、お客様の個人データを、正当な業務上の必要性がある者にのみ開示します。 当社は、お客様の個人データを開示する場合は常に、本プライバシー通知 (および個人データの機微度と分類に一致する、あらゆる適用される内部データ処理ガイドライン) と一致する方法で個人データが使用されることを確実にします。 お客様の個人データは、以下の正当な目的で、当社の子会社および関連会社やその他の第三者 (サービス プロバイダーを含みます) に開示される場合があります。

* 上記の個人データ処理の目的を実行するため (「当社が個人データを処理する理由」を参照してください)。
* 第三者が GitHub に代わってサービスを提供できるようにするため。
* 当社の法的義務、規制、政府機関の許可、もしくは契約を遵守するため、またはデータ主体の権利、裁判所命令、行政または訴訟手続き (召喚状、政府機関の監査、捜索令状など) に対応するため。 受領者のカテゴリには、契約の相手方当事者、司法機関、および政府機関が含まれます。
* 公的機関 (規制機関、法執行機関、国家安全保障機関など) による合法的な要請に応じる場合。
* 外部弁護士からの法的助言、および会計士、経営コンサルタントなどのその他の外部専門家からの助言を求めるため。
* 訴訟の可能性や恐れまたは実際の訴訟について立証、権利行使、または弁護を行うために必要な場合。
* GitHub、お客様または他者の重大な利益 (安全やセキュリティなど) を守るために必要な場合。
* その他、お客様の同意に従う場合。

法的要件によってお客様の個人データの開示が制限されている場合、GitHub は当該要件を尊重することにご注意ください。

### [お客様の個人データに対するお客様の権利](#your-rights-to-your-personal-data) ###

一部の地域において、お客様は、適用されるデータ保護法 (欧州連合および英国の一般データ保護規則など) に基づく一定の権利を有する場合があります。 地域/国別の追加情報については、本通知の[補遺](#addenda)を参照してください。

### [Cookie および Web ビーコンの使用](#use-of-cookies-and-web-beacons) ###

サイト ページでは、Cookie (デバイスに配置される小さなテキスト ファイル) や類似のテクノロジが使用される場合があります。 当社は、こうした Cookie や類似のテクノロジにより、お客様の好みや設定を保存および尊重すること、お客様がサインインできるようにすること、不正行為に対抗すること、ならびに当社の Web サイトやオンライン サービスのパフォーマンスを分析することができます。

また、当社は、Cookie の配信および使用状況とパフォーマンス データの収集に役立つ "Web ビーコン" も使用します。 当社の Web サイトには、第三者のサービス プロバイダーの Web ビーコン、Cookie、または類似のテクノロジが含まれる場合があります。

Cookie、Web ビーコン、および類似のテクノロジによって収集されるデータを制御するためのさまざまなツールがあります。 たとえば、お客様は、インターネット ブラウザーのコントロールを使用して、アクセスする Web サイトでの Cookie の使用方法を制限したり、Cookie を消去またはブロックすることでご自身の同意を撤回したりすることができます。

### [お客様の個人データのセキュリティ](#security-of-your-personal-data) ###
