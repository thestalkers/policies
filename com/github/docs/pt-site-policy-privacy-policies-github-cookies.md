Cookies do GitHub
==========

Exibir página como Markdown

[Cookies](#cookies)
==========

O GitHub oferece muita transparência sobre como usamos seus dados, como coletamos seus dados e com quem compartilhamos seus dados. Para esse fim, fornecemos esta página que detalha como usamos os cookies.

O GitHub usa cookies para fornecer e proteger nossos sites, bem como para analisar o uso de nossos sites, a fim de oferecer a você uma ótima experiência de usuário. Confira nossa [Política de Privacidade](/pt/github/site-policy/github-privacy-statement#our-use-of-cookies-and-tracking-technologies) se quiser mais informações sobre cookies e saber como e por que os usamos, bem como os dados pessoais relacionados aos cookies. Você pode alterar sua preferência sobre cookies não essenciais quando quiser seguindo [estas instruções](/pt/account-and-profile/setting-up-and-managing-your-personal-account-on-github/managing-personal-account-settings/managing-your-cookie-preferences-for-githubs-enterprise-marketing-pages).

Como o número e os nomes dos cookies podem mudar, a tabela abaixo pode ser atualizada periodicamente. Para receber notificações de atualizações desta lista, siga as instruções fornecidas em [Sobre notificações](/pt/account-and-profile/managing-subscriptions-and-notifications-on-github/setting-up-notifications/about-notifications).
