Cookie-файлы GitHub
==========

[Использование файлов cookie](#использование-файлов-cookie)
==========

GitHub обеспечивает большую прозрачность в отношении того, как мы используем ваши данные, как мы собираем ваши данные и с кем мы делимся вашими данными. С этой целью мы приводим на этой странице подробные сведения о том, как мы используем файлы cookie.

GitHub использует файлы cookie для предоставления и защиты наших веб-сайтов, а также для анализа использования наших веб-сайтов, чтобы обеспечить вам удобство использования. Ознакомьтесь с нашим [Заявлением о конфиденциальности](/ru/github/site-policy/github-privacy-statement#our-use-of-cookies-and-tracking-technologies), если вы хотите получить дополнительные сведения о файлах cookie и о том, как и почему мы используем их, а также связанные с ними персональные данные. Вы можете в любой момент изменить настройки необязательных файлов cookie, следуя [этим инструкциям](/ru/account-and-profile/setting-up-and-managing-your-personal-account-on-github/managing-personal-account-settings/managing-your-cookie-preferences-for-githubs-enterprise-marketing-pages).

Поскольку количество и названия файлов cookie могут меняться, приведенная ниже таблица может время от времени обновляться. Чтобы получать уведомления об обновлении этого списка, следуйте инструкциям, приведенным в разделе [Общие сведения об уведомлениях](/ru/account-and-profile/managing-subscriptions-and-notifications-on-github/setting-up-notifications/about-notifications).
