Cookies de GitHub
==========

View page as Markdown

[Cookies](#cookies)
==========

GitHub brinda una gran transparencia con respecto a cómo usamos sus datos, cómo recopilamos sus datos y con quién los compartimos. Con ese fin, hemos creado esta página, en la que se detalla la forma en que usamos las cookies.

GitHub utiliza cookies para proporcionar y proteger nuestros sitios web, así como para analizar el uso de nuestros sitios web, a fin de ofrecerle una excelente experiencia de usuario. Eche un vistazo a nuestra [Declaración de Privacidad](/es/github/site-policy/github-privacy-statement#our-use-of-cookies-and-tracking-technologies) si desea obtener más información sobre las cookies, sobre cómo y por qué las usamos y sobre qué datos personales están relacionados con las cookies. Puede cambiar sus preferencias en relación con las cookies no esenciales siempre que lo desee; para ello, siga [estas instrucciones](/es/account-and-profile/setting-up-and-managing-your-personal-account-on-github/managing-personal-account-settings/managing-your-cookie-preferences-for-githubs-enterprise-marketing-pages).

Dado que el número y los nombres de las cookies pueden cambiar, la siguiente tabla puede actualizarse de vez en cuando. Para recibir notificaciones sobre las actualizaciones en esta lista, siga las instrucciones proporcionadas en [Acerca de las notificaciones](/es/account-and-profile/managing-subscriptions-and-notifications-on-github/setting-up-notifications/about-notifications).
